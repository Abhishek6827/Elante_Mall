# Elante Mall

A static website project built using **HTML, CSS, and JavaScript**.

## 🚀 Live Demo
The project is hosted on **GitHub Pages** and can be accessed here:

👉 [Live Site](https://abhishek6827.github.io/Elante_Mall/)

## 📂 Project Structure
```
Elante_Mall/
 ├── index.html
 ├── css/
 ├── js/
 ├── img/
```

## 🛠️ Features
- Responsive design
- Interactive UI using JavaScript
- Carousel and testimonial section
- Clean folder structure for assets

## ⚡ Hosting
This project is hosted using **GitHub Pages**:
1. Go to **Settings → Pages**
2. Select `main` branch and root folder
3. Save and get your live link
